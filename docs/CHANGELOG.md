# Changelog

## [1.3.0] - 2026-03-18

### Added
- **Система званий**:
  - Автоматическое начисление званий за количество сообщений
  - Звания за победы в дуэлях
  - Команды `!звание`, `!ачивка`, `!ачивки`
  - Команда `!прогресс`
  - Команда `!установитьзвание`
  - Топ пользователей по званиям (`!топ звания`).
- **Кулдауны**:
  - Для `!рулетки`
  - Настраиваются в `config/cooldowns.js`.
- **Улучшенный антиспам**:
  - Нормализация сообщений.
  - Исключение стримера и модераторов из автомодерации.
- **Telegram-бот**:
  - Автоматическое появление меню с инлайн-кнопками при запуске.
  - Управление жалобами, банами, таймаутами, очками, предупреждениями прямо из Telegram.
  - Удаление старых сообщений после использования.
  - Логирование событий.
- **Конфигурационные файлы**:
  - `config/cooldowns.js`
  - `config/moderation.js`
  - `config/titles.js`
- **Команды**:
  - `!перевод` – перевод очков между пользователями.
  - `!+очки` – начисление очков (модераторы/стример).
  - `!итоги` – статистика стрима (нарушения, баны, таймауты).
  - `!топ iq`, `!топ айкью` – поддержка русскоязычных алиасов.
  - `!жалобы` – просмотр жалоб (модераторы).
  - `!сбросить_подарки` – сброс кэша подарков (стример).
  - `!тест` – тестовая команда для отладки.
- **API**:
  - Эндпоинт `/reports/resolve` для отметки жалоб как решённых.
  - Эндпоинт `/stats/user/:username` для получения статистики пользователя.
- **Рефакторинг**:
  - Выделены хендлеры `automod.js`, `copypasta.js`, `messageHandler.js`, `titleHandler.js`.
  - Все числовые параметры вынесены в конфиги.
  - Унифицирована сигнатура функций в `interactive.js` (`(channel, args, username, tags, isBroadcaster, isMod, bot)`).
  - Добавлено логирование в проблемных местах для упрощения отладки.
- **База данных**:
  - Автоматическое приведение имён пользователей к нижнему регистру во всех таблицах.
  - Исправлена функция `addUserPoints` (теперь без `ON CONFLICT`).
  - Исправлена функция `updateUserIQ` (теперь с логикой UPDATE + INSERT).
  - Добавлен скрипт `clean-db.js` для очистки дублей и мусора.

### Changed
- **Автомодерация**:
  - Фильтр ссылок теперь применяется только к обычным пользователям (исключены модераторы, стример, VIP).
  - Запрещённые слова обрабатываются с прогрессивными наказаниями (первое нарушение – таймаут, второе – удвоенный, третье – бан).
- **База данных**:
  - Таблица `bot_points` расширена полями `wins`, `messages_count`, `title`.
  - Все функции работы с БД теперь приводят `username` к нижнему регистру.
- **Логика команд**:
  - Улучшена обработка аргументов в `handleTop`, `handleAddPoints`, `handleTransfer`.
  - Команды, требующие модераторских прав, проверяют `isMod` / `isBroadcaster`.
- **Документация**: обновлён `README.md` и `CHANGELOG.md`.

### Fixed
- Исправлена ошибка `SQLITE_CONSTRAINT` в `addUserPoints` и `updateUserIQ`.
- Устранены дубли в топах (`!топ очки`, `!топ iq`, `!топ варн`) за счёт нормализации базы.
- Команда `!перевод` больше не позволяет переводить очки самому себе и корректно проверяет баланс.
- Команда `!+очки` теперь отвечает в чат и действительно начисляет очки.
- Команда `!установитьзвание` больше не вызывает ошибку `handleSetTitle is not a function`.
- Команда `!жалобы` доступна только модераторам и возвращает читаемый список.
- Команда `!итоги` в офлайне возвращает корректное сообщение.
- Исправлен конфликт портов Telegram-бота (ошибка 409) – добавлен принудительный kill процессов при запуске.
- Исправлена логика копипасты – теперь не срабатывает на сообщения с упоминаниями.
- Исправлены все синтаксические ошибки после рефакторинга.

### Known Issues
- В редких случаях может возникать ошибка `No response from Twitch` – рекомендуется перезапустить бота.
- При очень высокой нагрузке возможны задержки в обработке сообщений – оптимизация запланирована в следующих версиях.
