# Changelog

## [1.2.0] - 23-02-2026

### Added
- FUTURE.md
- viewerStatus.js
- Новые интерактивные команды:
  - `!предложение [игра]` – предложить игру для голосования
  - `!голос [игра]` – проголосовать за игру
  - `!итоги` – показать результаты голосования и статистику стрима
  - `!пословица [слово]` – случайная пословица с подстановкой слова
  - `!рулетка` – игровой автомат с выигрышем очков
  - `!баллы` / `!очки` – показать количество очков пользователя
  - `!сбросить_подарки` – сброс кэша подарков (только для стримера)
- Новые сервисы: 
    - `voting.js` (голосование)
    - `points.js` (очки бота)
    - `proverbs.js` (пословицы)
    - `roulette.js` (рулетка)
- Автоматическое создание недостающих колонок в БД (`iq`, `updated_at`)
- Вывод в чат причины таймаута/бана при модерации
- Для стримера и модераторов сняты ограничения на команду `!подарок`
- Система предупреждений за капс: два предупреждения → таймаут

### Changed
- Полностью переработана структура проекта
- Обновлён `.gitignore`
- Удалены неуместные ремарки разработчика
- Улучшена обработка ошибок и добавлены таймауты для API-запросов
- Обновлён `README.md` для версии 1.2.0
- Модернизирована автоматическая модерация (капс, спам)

### Fixed
- Исправлена ошибка `SQLITE_ERROR`
- Исправлена ошибка `points.addUserPoints is not a function`
- Команда `!рулетка` теперь корректно начисляет очки
- Команда `!подарок` больше не создаёт проблем для модераторов и стримера
- Устранены проблемы с путями
- Исправлено задвоение сообщений при автоматической модерации
- Корректная работа команды `!варн` при третьем предупреждении
