# Changelog

## [1.2.2] - 25-02-2026

### Added
- `jokes.js`
- `fortunes.js`
- `duels.js`
- `unban.js`
- `reports.js`
- Фильтр ссылок
- Система авторазбана

### Changed
- Обновление БД
- `points.js`: добавлены методы `transferPoints` для перевода очков и проверки баланса
- `moderation.js`: функция `handleBan` теперь сохраняет бан в БД; добавлена команда `!разбан` для модераторов
- `interactive.js`: обновлён список команд в `handleCommands`; добавлены обработчики новых команд
- `bot.js`: интегрированы новые сервисы, добавлена поддержка дуэлей (`duelManager`) и авторазбана (`unbanManager`)

### Fixed
- Исправлены конфликты слияния в `interactive.js`
- Устранены ошибки при отсутствии необходимых колонок в базе данных (автоматическое добавление через `ALTER TABLE`)
- Исправлена проблема с начислением очков при дуэлях – теперь победитель получает удвоенную ставку
- Корректная обработка `!перевод` при недостаточном балансе
