# Идеи развития:        // сидеть думать+реализовывать

1. !дуэль @ник [ставка]

2. !перевод @ник сумма

3. !топ [категория] – топ пользователей по очкам, IQ, количеству подарков
    Примеры: !топ очки, !топ iq, !топ подарки

4. !анекдот – случайный анекдот из базы (можно пополнять модераторами через !+анекдот) // запретить 4ukreev создавать анекдоты!!!

5. бонус – получить ежедневный бонус (например, 5 очков, раз в 24 часа для каждого зрителя), хранить в БД время последнего получения.
Можно реализовать через !подарок

6. !судьба — случайное предсказание из базы (пополняется модераторами)

7. !когда — показать время до следующего стрима (если известно) или случайную отговорку ("когда рак на горе свистнет")

8. фильтр ссылок // НАДО

9. авторазбан  // для стримера и модераторов, которые не будут разбанивать неугодных ручками через неделю/месяц/год

10. !репорт @ник [причина]
