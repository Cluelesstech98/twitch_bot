# Идеи развития:

1. !дуэль @ник [ставка]

2. !перевод @ник сумма

3. !топ [категория] – топ пользователей по очкам, IQ, количеству подарков
Примеры: !топ очки, !топ iq, !топ подарки

4. !монетка [ставка] – подбросить монетку (орёл/решка). При совпадении выигрыш = ставка × 2, иначе проигрыш

5. !анекдот – случайный анекдот из базы (можно пополнять модераторами через !+анекдот) // запретить 4ukreev создавать анекдоты!!!

6. !бонус – получить ежедневный бонус (например, 5 очков, раз в 24 часа для каждого зрителя), хранить в БД время последнего получения.
Можно реализовать через !подарок

7. !судьба — случайное предсказание из базы (пополняется модераторами)

8. !когда — показать время до следующего стрима (если известно) или случайную отговорку ("когда рак на горе свистнет")
